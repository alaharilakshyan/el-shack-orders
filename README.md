## 📖 Overview

**EL Shack Orders** is a modern full-stack web application designed to streamline the food ordering experience. It allows users to browse menu items, place orders, and manage their selections through an intuitive and responsive interface.

The platform focuses on providing a seamless ordering workflow with real-time interactions, clean UI, and efficient backend handling.

---

## ✨ Features

* 🛒 **Online Ordering System** – Browse menu and place orders easily
* 🔐 **User Authentication** – Secure login & signup functionality
* 📦 **Order Management** – Track and manage user orders
* ⚡ **Real-time Interaction** – Smooth and responsive UI
* 📱 **Responsive Design** – Works across mobile, tablet, and desktop
* 🎨 **Modern UI/UX** – Clean and user-friendly interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/alaharilakshyan/el-shack-orders.git
cd el-shack-orders
```

---

### 2️⃣ Install dependencies

#### For frontend

```bash
cd frontend
npm install
```

#### For backend

```bash
cd backend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file in backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run the application

#### Start backend

```bash
npm run dev
```

#### Start frontend

```bash
npm start
```
